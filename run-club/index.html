The
Runner
Mark
A minimal running figure in mid-stride — both feet off the ground, the body in the float phase that every runner recognises. Not a logo. A waymark. The thing you see at the start line and the finish.
Used at all sizes — from the wristband stamp to the event banner. Minimum 20px digital, 8mm print.
Dawu on dark. Volt on dark for high-energy moments. Usiku on Pumzi for light backgrounds.
Never filled, never dropped in a box, never rotated beyond ±3°.
